## E. 定性的結論の検証（#15 で変わったか）

| 結論 | 旧 | 新（WSL・マルチシード） | 判定 |
|---|---|---|---|
| 下流タスク AIM ≳ MST > INDEPENDENT | 維持 | exp_e ε=1: AIM 0.693 ≈ MST 0.690 > IND 0.510。exp_b: AIM 0.709 > MST 0.651 > IND 0.506（10seed・#20） | **不変**（AIM≳MST、両者>IND） |
| 相関誤差は MST が最良（ε=1） | MST 0.064 最良 | 10seed ε=1: AIM 0.022 < MST 0.073 < IND 0.076（AIM 最良）。ただし AIM の ε=1 相関は run-to-run 不安定（別実行 5seed では 0.104±0.163） | **更新（#20）**: 本実行では AIM 最良だが ε=1 は run 依存 |
| AIM の相関誤差は MST より大（ε=1） | AIM 0.226 > MST 0.064 | 10seed ε=1: AIM 0.022 < MST 0.073。**ε≥2 は安定して逆転** AIM 0.019/0.024 < MST 0.085/0.066 | **新知見: ε≥2 で安定して AIM<MST**（ε=1 は run 不安定） |
| AIM ε スイープは分散支配で**トレンド判定不能** | §7 の結論 | **判定可能に更新**: TVD は ε で単調改善、AUC は **ε=0.5→1.0 に実在する谷**（#20 の 10seed で 3 水準の分布が重ならず確定、残課題③解消） | **更新**（pending 解消・#13/#14/#20） |
| 1-way TVD は機構間で同程度 | 0.098–0.106 | 10seed ε=1: MST 0.090 / AIM 0.087 / IND 0.088（同程度） | **不変** |
| MIA: DP合成は漏洩なし・ε非依存 | AUC≈0.51 | AUC≈0.51（D表） | **不変** |

**結論: 定性的な逆転（headline 反転）は無い。** 主要な更新は (1) §7 が「判定不能」→「マルチシードで判定可能・ε=1 に実在する AUC の谷」、(2) AIM ε スイープが全機構のマルチシードで取得済み（fig2/§5.2 の "pending" 解消）、(3) 高 ε で AIM の相関誤差が MST を下回る新知見。**Issue #20 の更新**: AIM を 5→10 シードに拡張し 3 機構を共通シードの単一系列で再生成 — (a) ε=1 の AUC 谷は **10 シードで実在を確定**（残課題③解消）、(b) AIM の相関誤差は ε≥2 で**安定して** MST を下回る（ε=1 は run 依存で、本実行では AIM が最小）。
