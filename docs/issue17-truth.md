## E. 定性的結論の検証（#15 で変わったか）

| 結論 | 旧 | 新（WSL・マルチシード） | 判定 |
|---|---|---|---|
| 下流タスク AIM ≳ MST > INDEPENDENT | 維持 | exp_e ε=1: AIM 0.707 > MST 0.659 > IND 0.519。exp_b: MST 0.687 ≈ AIM 0.677 > IND 0.506 | **不変**（AIM≈MST、両者>IND） |
| 相関誤差は MST が最良（ε=1） | MST 0.064 最良 | マルチシード ε=1: MST 0.048 < IND 0.079 < AIM 0.104 | **不変**（MST 最良） |
| AIM の相関誤差は MST より大（ε=1） | AIM 0.226 > MST 0.064 | ε=1: AIM 0.104 > MST 0.048。**ただし ε≥2 では逆転** AIM 0.020 < MST 0.080 | ε=1 は不変。**新知見: 高εでAIM相関がMSTを下回る** |
| AIM ε スイープは分散支配で**トレンド判定不能** | §7 の結論 | **判定可能に更新**: TVD は ε で単調改善、AUC は **ε=0.5→1.0 に実在する谷**（exp_e 5seed・script13 10seed の両方で再現、|Δ|>合成std） | **更新**（pending 解消・#13/#14） |
| 1-way TVD は機構間で同程度 | 0.098–0.106 | マルチシード ε=1: MST 0.089 / AIM 0.103 / IND 0.091（同程度） | **不変** |
| MIA: DP合成は漏洩なし・ε非依存 | AUC≈0.51 | AUC≈0.51（D表） | **不変** |

**結論: 定性的な逆転（headline 反転）は無い。** 主要な更新は (1) §7 が「判定不能」→「マルチシードで判定可能・ε=1 に実在する AUC の谷」、(2) AIM ε スイープが全機構のマルチシードで取得済み（fig2/§5.2 の "pending" 解消）、(3) 高 ε で AIM の相関誤差が MST を下回る新知見。
